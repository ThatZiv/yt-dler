# Setup
1. Install [Node.JS](https://nodejs.org)
2. Install [FFMPEG](https://ffmpeg.zeranoe.com)
3. Download this repo
4. `cd` into the repo and do
```sh
$ npm install
``` 
5. Run `run.bat` or do 
```sh
$ node index.js
```
